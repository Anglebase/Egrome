Egrome

Egrome配置

Egrome的配置只需将include和lib目录下的内容复制到编译器(GCC，测试环境为GCC 13.2)的include和lib目录下即可，并在编译时额外添加-legrome参数即可。需要注意的是-legrome参数必须在-lgraphics64之前，否则会导致编译失败。

Egrome项目编译

在GCC编译器下，通过如下命令即可编译

g++ [源文件] -o [输出文件] -legrome -lgraphics64 -lgdiplus -luuid -lmsimg32 -lgdi32 -limm32 -lole32 -loleaut32 -lwinmm
