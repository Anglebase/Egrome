PainterEvent类

PainterEvent类是用于绘图事件传递的类，负责创建Painter对象并提供绘图相关的接口

注解
用户可以通过在绘图时间传递函数中如下的调用方式来创建Painter对象：
auto &painter = event.beginPaint(this); // 获取Painter对象
// 绘图代码
painter.drawLine(0, 0, 100, 100);
...
event.endPaint(); // 绘图结束时，必须调用该函数
