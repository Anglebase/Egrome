详细描述

应用程序类

注解
应用程序类提供了各种系统接口的封装，并管理应用程序的生命周期。它提供了例如读写剪贴板、获取设置窗口的大小、位置、标题、全局鼠标坐标等功能。其次，它创建的对象应当全局唯一，如果尝试创建第二个实例，将会抛出异常。
