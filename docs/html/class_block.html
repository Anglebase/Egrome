详细描述

Block是所有UI对象的基类，它提供了视图绘制、事件处理等基本功能

用户可以通过继承Block类来实现自己的UI对象，通过重写paint()函数来实现自己的绘制逻辑，通过重写update()函数来实现自己的更新逻辑

◆ paintEvent()

void Block::paintEvent ( const PaintEvent & event)
protectedvirtual

虚函数，用于绘制 Block

此函数内不应修改 Block 的数据状态，应采用 update() 函数来更新数据状态

此函数会遍历子 Block 并调用子 Block 的 paint() 函数绘制子 Block

参数

注解
子类重写该函数时应当调用父类的 paint() 函数

被 Button , 以及 Checkbox 重载.
