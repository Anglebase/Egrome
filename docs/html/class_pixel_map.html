详细描述

像素图类

像素图类用于绘制图像，并提供裁剪、复制以及混合等功能

注解
用户可以通过如下方式获取位图对象的Painter对象：
PixelMap bitmap(100, 100);
auto& painter = bitmap.beginPaint();
// 绘制图像
painter.drawLine(0, 0, 100, 100);
// 结束绘制
bitmap.endPaint(); // 绘图结束后，必须调用该函数
